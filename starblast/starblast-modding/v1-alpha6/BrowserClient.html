BrowserClient

The Browser Client Instance for supporting mod codes running in Browser Modding.

Constructor

new BrowserClient(options)

Parameters:
Name Type Description
options object options for calling the object.
Note that if both one property and its aliases exist on the object, the value of the main one will be chosen
Properties
Name Type Attributes Default Description
cacheECPKey boolean set to true if you want to reuse ECP Key for the next run, false otherwise
sameCodeExecution boolean <optional>
 false loading the same code will trigger the execution or not
**Note:** This feature only works when you call `loadCodeFromString`, `loadCodeFromLocal` or `loadCodeFromExternal` methods, and not during the auto-update process
crashOnException boolean <optional>
 false when tick or event function, or mod code execution fails, the mod will crash (true)
crashOnError boolean alias of the property options.crashOnException
logErrors boolean <optional>
 true game will log any errors or not
logExceptions boolean alias of the property options.logErrors
logMessages boolean <optional>
 true game will log any in-game logs or not
